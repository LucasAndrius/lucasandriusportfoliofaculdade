Keylogger/Screenlogger

Esse projeto consiste em um sistema de monitoramento que também é reconhecido por ser um malware, esse software tem capacidade de criar pastas armazenar logs de teclas clicadas pelo usuario, além da captura da imagem do computador do usuário. Após a captura das teclas e imagens o sistema utiliza o protocolo smtp para envio das informações por e-mail.

Esse projeto vem sendo atualização, em sua nova versão fiz a troca de libs para captura de teclas e imagens do usuário para que o sistema ficasse mais leve, modifiquei o sistema de envio com o protocolo smtp para armazenamento em nuvem e adicionei novas features como captura de audio e dados do sistema operacional do usuário.

Acredito que por conta da evolução desse projeto hoje ele se asemelha mais como um spyware do que um sistema de monitoramento convencional e por conta disto acredito que não seja mais possível exibir mais detalhes do projeto.

Alarme para filtro de água

Desenvolvi um alarme para meu filtro de água utilizando arduino, sensor de umidade e buzzer, por diversas vezes não sabiamos quando estava acabando a água do filtro, com este alarme conseguimos medir o níve para abastecer o filtro com mais recorrencia para que não falte água.
